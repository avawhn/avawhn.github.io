## 一、不可进行额外操作

### 1、props

**用于父组件向子组件传递值**，需要避免在子组件中修改 props 的传递的数据

组件引用关系如图：

{% asset_img props组件关系.png 组件引用关系 %}

DemoView

```html
<template>
    <h2>DemoView</h2>
    <ChildComponent :count="count" />
    <button @click="subCount">count - 1</button>
</template>

<script>
    import ChildComponent from "@/components/ChildComponent.vue";
    export default {
        components: { ChildComponent },
        data() {
            return {
                count: 0,
            };
        },
        methods: {
            subCount() {
                this.count--;
            },
        },
    };
</script>
```

ChildComponent

```html
<template>
    <h2>Child</h2>
    <p>child component count: {{ count }}</p>
</template>

<script>
    export default {
        props: {
            count: {
                type: Number,
                default: 10,
            },
        },
    };
</script>
```

### 2、provide/ inject

适用于**父组件向多层级或者深层级子组件**传递数据，避免一层一层传递数据，如图所示：

{% asset_img 依赖注入示例图.png 依赖注入示例图 %}

示例组件依赖引用关系如下：

{% asset_img 依赖注入组件引用关系.png 组件依赖关系 %}

RootView

```html
<template>
    <h2>RootView</h2>
    <p>RootView Count: {{ count }}</p>
    <button @click="subCount">count减一</button>
    <ComponentA />
</template>

<script>
    import ComponentA from "@/components/inject/ComponentA.vue";
    import { computed } from "vue";
    export default {
        components: { ComponentA },
        provide() {
            return {
                // 请注意这不会使注入保持响应性
                // count: this.count

                // 为保证注入方和供给方之间的响应性链接，我们需要使用 computed() 函数提供一个计算属性
                count: computed(() => this.count),
            };
        },
        data() {
            return {
                count: 0,
            };
        },
        methods: {
            subCount() {
                this.count--;
            },
        },
    };
</script>
```

ComponentA

```html
<template>
    <h2>ComponentA</h2>
    <ComponentB />
</template>

<script>
    import ComponentB from "@/components/inject/ComponentB.vue";
    export default {
        components: { ComponentB },
    };
</script>
```

ComponentB

```html
<template>
    <h2>ComponentB</h2>
    <p>ComponentB Count: {{ count }}</p>
</template>

<script>
    export default {
        inject: {
            count: {
                from: "count", // 当与原注入名同名时，这个属性是可选的
                default: -1,
            },
        },
    };
</script>
```

## 二、可进行额外操作

下面示例组件关系如下：

{% asset_img eventbus组件关系.png 组件引用关系 %}

### 1、$emit

DemoView

```html
<template>
    <h2>DemoView</h2>
    <p>DemoView Count: {{ count }}</p>
    <ComponentA @sub-count="subCount" />
</template>

<script>
    import ComponentA from "@/components/emit/ComponentA.vue";
    export default {
        components: { ComponentA },
        data() {
            return {
                count: 0,
            };
        },
        methods: {
            subCount(v) {
                this.count -= v;
            },
        },
    };
</script>
```

ComponentA

```html
<template>
    <h2>ComponentA</h2>
    <button @click="subCount">让DemoView中的Count减一</button>
</template>

<script>
    export default {
        methods: {
            subCount() {
                this.$emit("sub-count", 1);
            },
        },
    };
</script>
```

### 2、eventBus

#### 2.1 vue2

event-bus.js

```js
import Vue from "vue";
export const eventBus = new Vue();
```

DemoView

```html
<template>
    <h2>DemoView</h2>
    <componentA />
    <componentB />
</template>

<script>
    import ComponentA from "@/components/eventbus2/ComponentA.vue";
    import ComponentB from "@/components/eventbus2/ComponentB.vue";

    export default {
        components: { ComponentA, ComponentB },
    };
</script>
```

ComponentA

```html
<template>
    <h2>ComponentA</h2>
    <button @click="sendMsg">让ComponentB打印'hello world'</button>
</template>

<script>
    import { eventBus } from "./event-bus.js";
    export default {
        data() {
            return {};
        },
        methods: {
            sendMsg() {
                eventBus.$emit("sendMsg", { msg: "hello world" });
            },
        },
    };
</script>
```

ComponentB

```html
<template>
    <h2>ComponentB</h2>
    <p>{{ msg === '' ? '消息为空' : msg }}</p>
</template>

<script>
    import { eventBus } from "./event-bus.js";
    export default {
        data() {
            return {
                msg: "",
            };
        },
        methods: {
            receiveMsg(param) {
                this.msg = param.msg;
            },
        },
        mounted() {
            eventBus.$on('sendMsg', this.receiveMsg)
        },
        unmounted() {
            eventBus.$off("sendMsg", this.receiveMsg);
        },
    };
</script>
```

#### 2.2 vue3

vue3 中 "$on"，"$off" 和 "$once" 实例方法已被移除，组件实例不再实现事件触发接口，这里我们使用官方推荐的这个第三方库 [mitt](https://github.com/developit/mitt) 实现

main.js

```js
import "./assets/main.css";

import { createApp } from "vue";
import App from "./App.vue";
import router from "./router";
// 引入 mitt
import mitt from "mitt";

const app = createApp(App);

// 配置全局事件总线
app.config.globalProperties.$bus = mitt();

app.use(router);
app.mount("#app");
```

DemoView

```html
<template>
    <h2>DemoView</h2>
    <componentA />
    <componentB />
</template>

<script>
    import ComponentA from "@/components/eventbus/ComponentA.vue";
    import ComponentB from "@/components/eventbus/ComponentB.vue";

    export default {
        components: { ComponentA, ComponentB },
    };
</script>
```

ComponentA

```html
<template>
    <h2>ComponentA</h2>
    <button @click="sendMsg">让ComponentB打印'hello world'</button>
</template>

<script>
    export default {
        data() {
            return {};
        },
        methods: {
            sendMsg() {
                this.$bus.emit("sendMsg", { msg: "hello world" });
            },
        },
    };
</script>
```

ComponentB

```html
<template>
    <h2>ComponentB</h2>
    <p>{{ msg === '' ? '消息为空' : msg }}</p>
</template>

<script>
    export default {
        data() {
            return {
                msg: "",
            };
        },
        methods: {
            receiveMsg(param) {
                this.msg = param.msg;
            },
        },
        mounted() {
            this.$bus.on("sendMsg", this.receiveMsg);
        },
        unmounted() {
            this.$bus.off("sendMsg", this.receiveMsg);
        },
    };
</script>
```

## 三、获取实例

示例组件关系如下：

{% asset_img ref组件关系.png 组件引用关系 %}

### 1、ref / refs

DemoView

```html
<template>
    <h2>DemoView</h2>
    <button @click="sendMsg">调用ComponentB的receiveMsg</button>
    <componentB ref="cb" />
</template>

<script>
    import ComponentB from "@/components/ref/ComponentB.vue";

    export default {
        components: { ComponentB },
        methods: {
            sendMsg() {
                this.$refs.cb.receiveMsg("hello world");
            },
        },
    };
</script>
```

ComponentB

```html
<template>
    <h2>ComponentB</h2>
    <p>{{ msg === '' ? '消息为空' : msg }}</p>
</template>

<script>
    export default {
        expose: ["receiveMsg"],
        data() {
            return {
                msg: "",
            };
        },
        methods: {
            receiveMsg(msg) {
                this.msg = msg;
            },
        },
    };
</script>
```
